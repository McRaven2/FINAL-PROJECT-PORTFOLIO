# Personal Portfolio — Your Name

A responsive personal portfolio website built with **HTML5**, **Bootstrap 5**, and a custom `style.css`.

## Live demo
*(replace with your GitHub Pages URL after deployment)*
https://username.github.io/portfolio

## What’s included
- `index.html` — main single-page portfolio
- `style.css` — custom styles overriding Bootstrap
- `assets/` — folder for images (profile & project screenshots)
- `README.md` — this file

## Features & Requirements (Course)
- Semantic HTML5 structure (`header`, `nav`, `main`, `section`, `footer`, `article`, `aside`)
- Responsive design (mobile, tablet, desktop)
- Bootstrap 5 with custom overrides (>=20% style changes)
- Projects gallery with title, screenshot, description, and links
- Contact form (Formspree example) and social links
- Ready for deployment to GitHub Pages

## How to use
1. Update text and links in `index.html` (your name, email, project URLs, social links).
2. Put profile and project images in `assets/` (update `src` paths).
3. (Optional) Replace Formspree URL with your own form endpoint.
4. Commit to a public GitHub repo. Enable GitHub Pages in repo settings (branch `main` or `gh-pages`).
